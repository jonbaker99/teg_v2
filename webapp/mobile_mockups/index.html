Two directions for mobile

Same page (Latest Leaderboard) and the same chart, built two ways. App leans into a native feel — bottom tab bar, segmented controls, tables reflowed into tappable rows. Editorial keeps the printed-programme look and just makes it sing on a phone. Flip light/dark below. Open any frame full-screen on your phone via the links.

tip: if a frame is blank, run a local server or open the links — file:// blocks iframes in some browsers
