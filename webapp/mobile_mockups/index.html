Two directions for mobile

The same page (Latest Leaderboard) and the same chart, built two ways. App leans into a native feel — bottom tab bar, segmented controls, tables reflowed into tappable rows. Editorial keeps the printed-programme look and just makes it sing on a phone. Each page has its own Light / Dark pill in the corner.

📱 Best viewed in your browser. If a page looks frozen in a preview, tap the share icon → Open in Safari / Chrome. The light/dark toggle is now pure CSS (no JavaScript), so it works in any real browser.

Round entry — decided (Phase 3.3)

Phone score entry

Pattern A is the chosen design going forward. Fully interactive (fake TEG 50 · Round 1 data, 7 players, nothing is saved) — a sticky-keypad grid with a fixed 2–8 keypad (always the same screen position, so muscle memory holds across holes; 1 and 9–12 stay reachable as a smaller outlier row), voice entry (tap the mic, dictate a run of scores with the keyboard's built-in dictation, check the parsed preview, then fill), and an all-players / single-player view toggle.
