TEG design round 2

Three visual languages × three stylings each. Every page has a working light/dark toggle (top right); charts re-colour live from each styling's tokens. Pages are fully self-contained (vendored Plotly/Tailwind) — only Google Fonts loads from the network.

Theme A — Plain · “the reference book”

No cards, no shadows, no fills. Hierarchy from typography, hairline rules and alignment; tables read like a well-set almanac. Accent spent only on interactive state.
